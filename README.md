# :construction_worker: How to run
### Rename env file
Rename `.env-exemple` to `.env`
### Install Dependencies
```bash
yarn install
```
### Run Aplication
```bash 
yarn dev 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
